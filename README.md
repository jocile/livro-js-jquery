# livro-js-jquery

===============

Código relativo ao livro:
Dominando Javascript com JQuery

Que poderá ser adquirido no site : url do site

## Descrição

Este repositório contém os códigos-fonte e exemplos práticos desenvolvidos ao longo do livro "Dominando JavaScript com jQuery". O material serve de apoio para o aprendizado prático das tecnologias abordadas, cobrindo desde os fundamentos da linguagem JavaScript até o uso avançado da biblioteca jQuery para criação de interfaces web ricas e interativas.

## Tecnologias Abordadas

- **JavaScript**: Conceitos essenciais, manipulação do DOM e boas práticas.
- **jQuery**: Seletores, eventos, animações e simplificação de scripts.
- **AJAX e JSON**: Comunicação assíncrona com o servidor.
- **jQuery UI e Mobile**: Componentes de interface e desenvolvimento para dispositivos móveis.

## Como Utilizar

Para visualizar os exemplos, você pode clonar este repositório e abrir os arquivos `.html` de cada capítulo diretamente em seu navegador web. Note que alguns exemplos específicos, como os que utilizam AJAX (Capítulos 7 e 8), podem exigir um servidor web local para funcionarem corretamente devido às políticas de segurança dos navegadores (CORS).
